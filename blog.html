What is DOM and it's importance

DOM means Document Object Model. It is used to manipulate the content, style and structure of a document using Javascript.

How to select element from DOM?

There are five ways by which elements can be selected from DOM. They are:
1. getElementsByTagName()
2. getElementsByClassName()
3. getElementById()
4. querySelector()
5. querySelectorAll()

What is Callback Function and how it is used?
